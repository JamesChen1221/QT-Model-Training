## 📊 如何解讀結果

### 預測值意義

| 預測值 | 意義 | 建議 |
|--------|------|------|
| +10% 以上 | 非常有潛力 | 重點關注 |
| +5% ~ +10% | 有潛力 | 可以考慮 |
| +2% ~ +5% | 一般 | 謹慎評估 |
| 0% ~ +2% | 潛力較小 | 觀望 |
| 負數 | 預期下跌 | 避免做多 |

### 檔案格式要求

```

  錯誤: 60.9, 63.3, 54.6, 47.2, 48.3

產業代碼:
  數字即可: 10, 22, 36, 37, 44

觸發類型:
  1 = 財報
  2 = 新聞
  3 = 財報 + 新聞

展望:
  1 = 調升
  0 = 持平
  -1 = 調降
```
