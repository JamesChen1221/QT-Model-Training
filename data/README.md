# 資料目錄

此目錄用於存放訓練和預測所需的資料檔案。

## 📁 檔案說明

### 訓練資料
- **QT Training Data.xlsx** - 訓練資料（未包含在 Git 中）
  - 工作表1: 訓練資料
  - 工作表2: 欄位說明
  - 工作表3: 產業代碼對照表

### 預測資料
- **Stock TBP.xlsx** - 待預測股票資料（未包含在 Git 中）

### 預測結果
- **new_predictions.csv** - 預測結果（自動生成，未包含在 Git 中）

## 📋 資料格式

### QT Training Data.xlsx 格式

#### 必要欄位

**基本資訊**：
- 開盤日期
- 公司代碼
- 產業（產業代碼，如 10, 11, 12...）

**價格距離指標**：
- 5日高價距離 (%)
- 5日低價距離 (%)
- 1個月高價距離 (%)
- 1個月低價距離 (%)
- 6個月高價距離 (%)
- 6個月低價距離 (%)

**盤前資訊**：
- 盤前 (%)

**觸發與情緒**：
- 觸發類型（1=財報為主, 2=消息為主, 3=財報+消息）
- 消息情緒分數

**財報資訊**：
- EPS Surprise (%)
- Revenue Surprise (%)
- 展望 (Guidance)（1=調升, 0=持平, -1=調降）

**趨勢特徵**：
- 120天收盤價序列（格式：`[100.5, 102.3, 101.8, ...]`，共120個值）

**目標標籤（帶 `#` 前綴）**：
- #開盤 (%)
- #10分鐘低價 (%)
- #1.5小時高價 (%)
- #最高價前的最低價 (%)

#### 可選欄位（帶 `*` 前綴，會被忽略）

- *昨日收盤價
- *盤前 10 分鐘價格
- *開盤價格
- *10分鐘最低價
- *1.5小時最高價
- *最高價前的最低價
- *進場位置 (%)
- *忍受回檔位置 (%)
- *獲利位置 (%)
- *獲利能力 (%)
- *備註
- *5天 RSI 序列
- *5天 ADX 序列
- *1個月 RSI 序列
- *1個月 ADX 序列
- *6個月 RSI 序列
- *6個月 ADX 序列

### Stock TBP.xlsx 格式

與訓練資料相同，但**不需要**目標標籤（`#` 開頭的欄位）。

## 🔒 資料隱私

Excel 資料檔案已加入 `.gitignore`，不會被上傳到 Git 倉庫。

如需使用此專案：
1. 準備你自己的訓練資料（`QT Training Data.xlsx`）
2. 準備待預測資料（`Stock TBP.xlsx`）
3. 放入 `data/` 目錄
4. 執行訓練和預測
