## 📋 上傳步驟

### 步驟 1: 在 GitHub 創建新倉庫

1. 前往 https://github.com/new
2. 填寫資訊：
   ```
   Repository name: qt-daytrading-prediction
   Description: QT 當沖潛力預測系統 - 使用 XGBoost 預測股票當沖潛力
   Public/Private: 選擇你想要的
   不要勾選 "Initialize this repository with a README"
   ```
3. 點擊 "Create repository"

### 步驟 2: 連接遠端倉庫

複製 GitHub 給你的 URL，然後執行：

```bash
git remote add origin https://github.com/你的用戶名/qt-daytrading-prediction.git
```

### 步驟 3: 推送到 GitHub

```bash
git push -u origin main
```

### 步驟 4: 驗證上傳

前往 GitHub 倉庫頁面，確認：
- [ ] 程式碼檔案都已上傳
- [ ] Excel 檔案沒有被上傳
- [ ] README.md 正確顯示
- [ ] data/README.md 存在

## 🔍 最終檢查

執行以下命令確認：

```bash
# 1. 檢查當前分支
git branch
# 應該顯示 * main

# 2. 檢查 Git 狀態
git status
# 應該顯示 "nothing to commit, working tree clean"

# 3. 檢查沒有 Excel 檔案被追蹤
git ls-files | grep -i xlsx

# 4. 確認本地 Excel 檔案仍存在

# 5. 檢查提交歷史
git log --oneline -5
# 應該看到最近的提交記錄
```
