# 專案結構

## 📁 檔案清單

```
QT Model Training/
│
├── 📂 data/                                    # 資料目錄
│   ├── QT Training Data.xlsx                   # 訓練資料（17筆）

│
├── 📂 models/                                  # 模型目錄
│   ├── qt_model_開盤_pct.pkl                    # 開盤漲幅預測模型

│   ├── qt_model_1.5小時高價_pct.pkl             # 1.5小時高價預測模型
│   ├── qt_model_最高價前的最低價_pct.pkl         # 最低價預測模型
│   ├── training_results_開盤_pct.png            # 訓練結果圖表

│   ├── training_results_1.5小時高價_pct.png     # 訓練結果圖表
│   └── training_results_最高價前的最低價_pct.png # 訓練結果圖表
│
├── 🐍 train_qt_xgboost.py                      # 訓練腳本（核心程式）
├── 🐍 predict_new_stocks.py                    # 預測腳本（核心程式）
│
├── ⚡ 快速訓練.bat                              # 一鍵訓練工具
├── ⚡ 快速預測.bat                              # 一鍵預測工具
│
├── 📄 requirements.txt                         # Python 依賴套件
├── 🚫 .gitignore                               # Git 忽略規則
│
└── 📚 說明文件/
    ├── README.md                               # 專案說明（主要文檔）
    ├── 專案結構.md                              # 本文件
    ├── 訓練概念說明.md                          # 機器學習概念
    ├── 斜率計算方式說明.md                      # 趨勢斜率特徵說明
    ├── 欄位標記說明.md                          # Excel 欄位命名規則
    ├── 觸發類型處理說明.md                      # 觸發類型邏輯
    └── 趨勢斜率特徵提取總結.md                  # 特徵提取方法總結
```

---

## 🎯 核心檔案（不可刪除）

### 程式碼
- **train_qt_xgboost.py** - 訓練 4 個 XGBoost 模型
- **predict_new_stocks.py** - 使用 4 個模型進行預測

### 資料
- **data/QT Training Data.xlsx** - 訓練資料（17筆記錄）
- **data/Stock TBP.xlsx** - 待預測股票資料

### 工具
- **快速訓練.bat** - 一鍵訓練（Windows）
- **快速預測.bat** - 一鍵預測（Windows）

### 配置
- **requirements.txt** - Python 依賴套件清單
- **.gitignore** - Git 版本控制配置

---

## 🔄 自動生成的檔案（可刪除）

這些檔案會在使用時自動生成：

- **data/new_predictions.csv** - 預測結果（每次預測時生成）
- **models/qt_model_*.pkl** - 模型檔案（訓練時生成）
- **models/training_results_*.png** - 訓練圖表（訓練時生成）

## 🚀 快速使用

### 訓練模型
```bash
# Windows
雙擊 快速訓練.bat

# 命令列
python train_qt_xgboost.py
```

### 預測股票
```bash
# Windows
雙擊 快速預測.bat

# 命令列
python predict_new_stocks.py --input "data/Stock TBP.xlsx"
```
