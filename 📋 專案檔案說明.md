# 📋 專案檔案說明

## 🎯 核心檔案（必須保留）

### 訓練與預測
- **train_qt_xgboost.py** - 訓練模型的主要腳本
- **predict_new_stocks.py** - 預測新股票的腳本
- **快速預測.bat** - 一鍵預測工具（最方便）

### 資料
- **data/QT Training Data.xlsx** - 訓練資料（你的資料）
- **single stock.xlsx** - 預測範例

### 模型
- **models/qt_xgboost_model.pkl** - 訓練好的模型
- **models/training_results.png** - 訓練結果圖表

### 配置
- **requirements.txt** - Python 依賴套件清單
- **config.py** - 模型配置參數
- **.gitignore** - Git 忽略規則

### 說明文件
- **README.md** - 完整專案說明
- **✅ 預測成功！下次這樣做.md** - 快速操作指南
- **訓練概念說明.md** - 機器學習概念

---

## 🔄 自動生成的檔案（可刪除）

這些檔案會在使用時自動生成：

## 🚀 快速使用

### 訓練模型
```bash
python train_qt_xgboost.py
```

### 預測股票
```bash
# 最快方式
雙擊 快速預測.bat

# 命令列方式
python predict_new_stocks.py --input "single stock.xlsx"
```

### 查看說明
- **README.md** - 完整說明
- **✅ 預測成功！下次這樣做.md** - 快速指南
